# AGENTS.md

Backend for a portfolio website — Express 5 · TypeScript · Mongoose · Zod · JWT · Multer + Cloudinary.

**See `CLAUDE.md` for the full architecture, module conventions, middleware patterns, and environment variable reference.**

## Commands

```bash
npm run dev    # development with hot reload
npm run build  # compile TypeScript → dist/
npm start      # run compiled output
```

No test runner is configured.
